The Genome

Essential Understanding

The genome is all your DNA collectively. Most complex organisms have their genome split up into multiple chromosomes, with humans having two copies of each chromosome.

Genome Organization

Structure

  • Complete set of genetic material
  • Multiple chromosomes
  • Two copies of each chromosome
  • One copy from each parent

Human Genome

  • 23 chromosome pairs
  • About 3 billion base pairs
  • Approximately 20,000 genes
  • Large amounts of non-coding DNA

Genes and Alleles

Gene Structure

  • Specific DNA sequences
  • Code for proteins/traits
  • Include regulatory regions
  • Can have multiple versions (alleles)

Allele Variations

  • Different versions of same gene
  • Result from DNA mutations
  • Create genetic diversity
  • Influence traits/phenotypes

Genome Function

Key Roles

  • Blueprint for development
  • Control of cell activities
  • Response to environment
  • Inheritance of traits

Beyond Genes

  • Regulatory sequences
  • Non-coding RNA genes
  • Structural elements
  • Mobile genetic elements

Quick Check

Why do organisms have two copies of each chromosome?

Having two copies (one from each parent) provides backup if one copy has mutations, allows for genetic diversity through different allele combinations, and enables evolution through genetic variation.
