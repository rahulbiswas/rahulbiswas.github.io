The Genetic Code

Essential Understanding

The genetic code is the set of rules by which DNA sequences are translated into amino acid sequences, using three-letter codons to specify each amino acid.

Code Properties

Key Features

  • Three-letter codons (triplets)
  • Non-overlapping code
  • Universal across most species
  • Degenerate (redundant)

Special Codons

  • Start codon (AUG)
  • Stop codons (UAA, UAG, UGA)
  • Reading frame importance
  • Signal sequences

Codon Tables

Organization

  • 64 possible codons
  • 20 amino acids coded
  • Multiple codons per amino acid
  • Codon families

Wobble Rules

  • Third position flexibility
  • tRNA recognition
  • Reduces tRNA number needed
  • Evolution implications

Evolution

Code Origin

  • Early life development
  • Chemical constraints
  • Selection pressure
  • Error minimization

Variations

  • Mitochondrial code
  • Nuclear differences
  • Species variations
  • Code expansion

Medical Applications

Clinical Relevance

  • Genetic diseases
  • Mutation effects
  • Drug development
  • Protein engineering

Quick Check

Why is the genetic code called "degenerate"?

The code is degenerate because multiple different codons can specify the same amino acid, providing redundancy that helps protect against some types of mutations.
