DNA & Transcription

1. If a DNA sequence reads ATGGCCTAA:

  • What is the complementary DNA strand?
  • What is the mRNA sequence?
  • How many amino acids will the protein have?
  • Complementary DNA: TACCGGATT
  • mRNA: AUGGCCUAA
  • Amino acids: 3 (ATG = start, GCC = Ala, TAA = stop)

2. Calculate the number of hydrogen bonds in this sequence:

GCATTAGC

GC = 3 bonds, AT = 2 bonds

G-C (3) + C-G (3) + A-T (2) + T-A (2) + T-A (2) + A-T (2) + G-C (3) + C-G (3) = 20 bonds

Translation Problems

3. Using the genetic code table, translate:

mRNA: AUG CCU GAC UAG

  • AUG = Methionine (Start)
  • CCU = Proline
  • GAC = Aspartic Acid
  • UAG = Stop
  • Final protein: Met-Pro-Asp

4. A mutation changes CUU to CUA. Is this a:

  • Silent mutation?
  • Missense mutation?
  • Nonsense mutation?

Silent mutation - both CUU and CUA code for Leucine

Gene Expression

5. Compare gene expression in these cells:

  • Active muscle cell
  • Developing neuron
  • Mature red blood cell
  • Muscle: High expression of contractile proteins
  • Neuron: Expression of neurotransmitter genes
  • RBC: No nucleus, no new gene expression
