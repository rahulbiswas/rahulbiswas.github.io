Multiple Choice

1. What is the first step in gene expression?

  • a) Translation
  • b) Transcription
  • c) Protein folding
  • d) RNA processing

b) Transcription - Gene expression begins with transcribing DNA into RNA.

2. Which mutation type changes the reading frame?

  • a) Silent mutation
  • b) Missense mutation
  • c) Frameshift mutation
  • d) Point mutation

c) Frameshift mutation - Insertions or deletions that alter the three-letter codon grouping.

Genetic Code Practice

5. Translate this mRNA sequence: AUG CCU GAA UAA

  • AUG = Methionine (Start)
  • CCU = Proline
  • GAA = Glutamic acid
  • UAA = Stop
  • Final protein: Met-Pro-Glu

Concept Application

6. How might a mutation in a transcription factor affect gene expression?

A mutated transcription factor might:

  • Fail to bind DNA properly
  • Affect multiple target genes
  • Disrupt developmental patterns
  • Lead to disease conditions
