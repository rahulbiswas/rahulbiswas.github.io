Multiple Choice

1. Which sequence represents the correct order of signal transduction?

  • a) Response ‚Üí Reception ‚Üí Transduction
  • b) Reception ‚Üí Response ‚Üí Transduction
  • c) Reception ‚Üí Transduction ‚Üí Response
  • d) Transduction ‚Üí Reception ‚Üí Response

c) Reception ‚Üí Transduction ‚Üí Response - Signals must first be received, then transmitted through the cell before responses occur.

2. What distinguishes RTKs from GPCRs?

  • a) Location in membrane
  • b) Enzymatic activity
  • c) Signal molecules
  • d) Response speed

b) Enzymatic activity - RTKs have intrinsic kinase activity, while GPCRs work through G proteins.

Short Answer

3. Compare local and long-distance signaling:

  • Local: short range, paracrine/autocrine
  • Long-distance: hormones, blood transport
  • Different molecule types used
  • Different response times

4. Explain signal amplification in cell signaling:

  • Enzyme cascades multiply signal
  • Each step activates multiple molecules
  • Second messengers increase quickly
  • Results in large cellular response
