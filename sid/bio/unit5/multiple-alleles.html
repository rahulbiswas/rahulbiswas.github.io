Multiple Alleles

Essential Understanding

Multiple alleles occur when a gene has more than two possible alleles in a population, though individuals still only carry two alleles for each gene.

Basic Concepts

Key Features

  • More than two alleles
  • Population-level variation
  • Individual diploid state
  • Complex inheritance

Inheritance Rules

  • Two alleles per individual
  • Dominance hierarchies
  • Multiple combinations
  • Population diversity

Quick Check

Why can a person have blood type AB if they only inherit two alleles?

While there are three possible alleles for blood type in the population (IA, IB, i), each person inherits only two. Type AB results from inheriting both IA and IB alleles, which are codominant.
